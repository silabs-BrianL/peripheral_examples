switch_led_polled

This project demonstrates a simple polled-mode use of the device's GPIOs.
Pushbuttons 0 and 1 are constantly polled. If pressed, LED0 and LED1, 
respectively, are turned on.

How To Test:
1. Build the project and download to the Starter Kit
2. Press PB0 to turn LED0 on
3. Press PB1 to turn LED1 on

Peripherals Used:
HFRCO  - 14 MHz

Board:  Silicon Labs EFM32G Starter Kit (Gxxx_STK)
Device: EFM32G890F128
PC0 - LED0
PC1 - LED1
PB9 - Push Button PB0
PB10 - Push Button PB1

pcnt_extclk_single_overflow

This project demonstrates the single external clock input oversampling
mode of the pulse counter with interrupts. In this example, Push Button
PB0 is the clock source and each clock cycle is generated by the press 
and release of the button. The program generates an interrupt whenever the 
pulse counter detects the number of pulses that went above a threshold set by user.

How To Test:
1. Build the project and download to the Starter Kit
2. Press Button 0 eight times
3. Observe LED0 toggle
4. Press Button 0 six times for any future tries
5. Observe LED0 toggle

When first run, an interrupt is requested after button 0 is pressed 8 times,
after which it is requested every 6 presses.  This happens because with a low
frequency clock, it needs to be synced for two clock cycles before the PCNT will
take the input, which in this case, is equal to two button presses.

For Gecko (EFM32Gxxx):

 This project demonstrates the single external clock input oversampling
 mode of the pulse counter with interrupts. In this example, Push Button
 PB0 is the clock source and each clock cycle is generated by the press 
 and release of the button. The program generates an interrupt whenever the 
 pulse counter detects the number of pulses that went above a threshold set by user.

 Note for Gecko Gxxx devices, the PRS cannot be used with the PCNT

Prepare to Test:
1. Wire PC5 on the expansion header(pin 5) to PE0 on the breakout pad
OR connect a function generator to PE0. Technically, the function
generator can be ran at any frequency. It is recommended to run the
generator under 10 Hz so you can track the number of pulses.

How To Test:
1. Build the project and download to the Starter Kit
3. Press Button 17 times (6 times to sync the clock)
4. Observe LED0 toggle
5. Press Button 0 12 times (equivalent to 6 pulses)
6. Observe LED0 toggle

When first testing code, Button 0 needs to be pressed 17 times to
generate enough pulses to sync the external clock. After that there is no
need to press 6 extra times. In addition, with the negative logic and the
prs connection, push button 0 will have a default 1 press after initialization.
Therefore for you only need to press the button 11+6 times to generate a total
of 9 pulses.
